# 1D RGB Game
This marvelous game is very fun to play! Check it out!

Using WS2812 RGB LED's.

# Setup
1. Install Arduino IDE
2. Download Adafruit Neopixel
3. Upload to supported device
4. Play with 'A' to shoot or 'D' to change color
